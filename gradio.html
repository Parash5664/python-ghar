About This App

This is the Gradio interface for Python Ghar, an AI-powered Python learning platform. The app provides:

  • AI Code Generation
  • Code Explanation
  • Interactive Code Execution
  • Learning Resources

How to Run Locally

To use this application locally:

  1. Clone the repository: git clone https://github.com/Parash5664/python-ghar.git
  2. Navigate to the project directory: cd python-ghar
  3. Install Python dependencies: pip install -r requirements.txt
  4. Run the Gradio app: python gradio_app.py
  5. Open your browser and go to: http://localhost:7863

Important Note

Gradio applications cannot be directly deployed to GitHub Pages because they require a Python backend server. GitHub Pages only serves static files.

To use the full Gradio application:

  • Run it locally using the instructions above
  • Deploy it to Hugging Face Spaces (recommended for Gradio apps)
  • Deploy it to a cloud platform like Heroku, Render, or Railway
